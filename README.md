# ShareEase-File-Sharing


## Clone the project

```bash
  git clone https://github.com/Pravesh-MW/ShareEase-File-Sharing
```


## Run Server

Go to the project directory

```bash
  cd server
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  node server.js
```
OR
```bash
  nodemon server.js
```
## Run Client

Go to the project directory

```bash
  cd client
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** React, TailwindCSS

**Server:** Node, Express, Multer, Mongodb
